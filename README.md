# MocoLoco

### MocoLoco project

#### How to compile

`g++ -Wall -o test -Wwrite-strings MocoLoco.cpp`

#### How to run 

`./test -b <file.bed>  -p <number> -t <file.2bit> -j <file.JASPAR>`-ss(optional)

#### this script uses the twobit.c and twobit.h file from
https://github.com/andrelmartins/TwoBit
